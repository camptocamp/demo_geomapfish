GeoMapFish demo project
=======================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.5/>`_

Checkout
--------

.. code::

   git clone git@github.com:camptocamp/demo_geomapfish.git
   cd demo_geomapfish

Build
-----

.. code::

  make secrets
  ./build env.default env.project env.secrets

Run locally
-----------

.. code::

  docker-compose up -d
