demo_geomapfish project
=======================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.4/>`_

Checkout
--------

.. code:: bash

   git clone git@github.com:camptocamp/demo_geomapfish.git

   cd demo_geomapfish

Build
-----

.. code:: bash

  make secrets

  ./docker-run make build

Run locally
-----------

.. code:: bash

  docker-compose up -d
