demo_geomapfish project
===================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.8/>`_

Checkout
--------

.. code::

   git clone git@github.com:camptocamp/demo_geomapfish.git

   cd demo_geomapfish

Build
-----

.. code::

  ./build

Run
---

.. code::

   docker-compose up -d
